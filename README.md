# Kişisel Harcama Takip Uygulaması  
Python • CSV • CLI Uygulaması • Raporlama Modülü

Bu proje, kullanıcıların günlük gelir ve giderlerini kaydedebildiği, aylık ve genel raporlar oluşturabildiği Python tabanlı bir kişisel harcama takip uygulamasıdır.

Uygulama verileri CSV formatında saklar ve tamamen komut satırı (CLI) üzerinden çalışır.


# Özellikler
- Gelir ve gider ekleme  
- Otomatik tarih atama  
- Kategori seçme menüsü  
- Açıklama ekleme  
- Tüm kayıtları listeleme  
- Genel rapor: toplam gelir, gider, net bakiye  
- Aylık rapor: belirli bir ay için gelir-gider analizi  
- En çok harcama yapılan kategoriyi gösterme  


#Kullanılan Teknolojiler
- **Python**  
- **CSV** dosya yönetimi  
- **datetime** ile tarih işlemleri  
- CLI tabanlı interaktif menü
